# AutoBlogAI 🤖📝

مشروع أندرويد متكامل للنشر التلقائي في مدونات Blogger باستخدام الذكاء الاصطناعي. يقوم التطبيق بجلب المواضيع من خلاصات RSS، إعادة كتابتها باحترافية باستخدام Gemini AI، توليد صور مناسبة، ثم نشرها تلقائياً.

## المميزات ✨
- **جلب المقالات من RSS:** جلب المقالات من خلاصات RSS فعلية بدلاً من البيانات الوهمية.
- **إعادة كتابة المحتوى:** تحويل النصوص العادية إلى مقالات احترافية محسنة للسيو (SEO) مع استخراج عنوان ومحتوى منفصلين.
- **توليد الصور:** جلب صور عالية الجودة من Pexels API.
- **نشر تلقائي:** التكامل مع Blogger API عبر OAuth 2.0 وتحديث حالة النشر بشكل صحيح.
- **عامل خلفي (Background Worker):** يعمل التطبيق في الخلفية كل 15 دقيقة لفحص ونشر الجديد، مع معالجة أخطاء محسنة.
- **تخزين آمن:** حفظ مفاتيح API بشكل مشفر باستخدام EncryptedSharedPreferences.
- **تحكم يدوي:** إمكانية تشغيل عملية النشر يدوياً من واجهة المستخدم.

## المتطلبات التشغيلية 🛠️
تحتاج إلى المفاتيح التالية لتشغيل المشروع:
1. **Gemini API Key:** من Google AI Studio.
2. **Blogger OAuth 2.0 Credentials:** (Client ID, Client Secret, Refresh Token, Blog ID).
3. **Pexels API Key:** لجلب الصور.
4. **RSS Feed URL:** رابط خلاصة RSS لجلب المقالات.

## الهيكل التقني 🏗️
- **اللغة:** Kotlin
- **واجهة المستخدم:** Jetpack Compose
- **الخلفية:** WorkManager
- **الشبكات:** OkHttp, Retrofit, Google API Client, SimpleXMLConverterFactory
- **قاعدة البيانات:** JSON Local Storage

## كيفية الاستخدام 🚀
1. قم بفتح المشروع في Android Studio.
2. توجه إلى شاشة الإعدادات وأدخل جميع مفاتيح API المطلوبة ورابط خلاصة RSS.
3. قم بحفظ الإعدادات، وسيبدأ التطبيق بالعمل تلقائياً في الخلفية، أو يمكنك تشغيله يدوياً من شاشة لوحة التحكم.
