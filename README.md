# AutoBlogAI 🤖📝

مشروع أندرويد متكامل للنشر التلقائي في مدونات Blogger باستخدام الذكاء الاصطناعي. يقوم التطبيق بجلب المواضيع من خلاصات RSS، إعادة كتابتها باحترافية باستخدام Gemini AI، توليد صور مناسبة، ثم نشرها تلقائياً.

## المميزات ✨
- **إعادة كتابة المحتوى:** تحويل النصوص العادية إلى مقالات احترافية محسنة للسيو (SEO).
- **توليد الصور:** جلب صور عالية الجودة من Pexels API.
- **نشر تلقائي:** التكامل مع Blogger API عبر OAuth 2.0.
- **عامل خلفي (Background Worker):** يعمل التطبيق في الخلفية كل 15 دقيقة لفحص ونشر الجديد.
- **تخزين آمن:** حفظ مفاتيح API بشكل مشفر باستخدام EncryptedSharedPreferences.

## المتطلبات التشغيلية 🛠️
تحتاج إلى المفاتيح التالية لتشغيل المشروع:
1. **Gemini API Key:** من Google AI Studio.
2. **Blogger OAuth 2.0 Credentials:** (Client ID, Client Secret, Refresh Token).
3. **Pexels API Key:** لجلب الصور.

## الهيكل التقني 🏗️
- **اللغة:** Kotlin
- **واجهة المستخدم:** Jetpack Compose
- **الخلفية:** WorkManager
- **الشبكات:** OkHttp, Retrofit, Google API Client
- **قاعدة البيانات:** JSON Local Storage

## كيفية الاستخدام 🚀
1. قم بفتح المشروع في Android Studio.
2. توجه إلى شاشة الإعدادات وأدخل جميع مفاتيح API المطلوبة.
3. قم بحفظ الإعدادات، وسيبدأ التطبيق بالعمل تلقائياً في الخلفية.
